# ファイアーエムブレムダメージ計算ツール

## 概要

ファイアーエムブレムの戦闘計算を再現するWebアプリです。  
ダメージ・命中・必殺・追撃などをまとめて確認でき、戦闘結果の目安を把握することを目的として作成しました。
現在、暗黒竜と光の剣、封印の剣のツールの実装を行いました。

---

## 機能

* ダメージ計算（特効・三すくみ対応）
* 命中率計算（実効命中率採用）
* 必殺率計算
* 追撃判定（攻速差）
* 支援補正（全属性対応）
* 地形補正
* 特殊武器対応

---

## 使用技術

* HTML
* CSS
* JavaScript（Vanilla JS）

---

## 工夫した点

* 実際のゲーム仕様に基づき、内部の計算を製作しました。
* 支援効果を全属性・全ランクでデータ化し、柔軟に計算できるようにしました。
* ダメージ・命中・必殺・追撃をそれぞれ独立した関数として分離し、可読性と保守性を意識しました。
* 地形や武器特性など複雑な条件分岐を整理し、仕様の再現性を高めました。
* ツールであるという点からあえてUIはシンプルにして硬派にしています。
* 使用者の負担をなるべく減らすために、入力量をできるだけ減らせるように意識しました。
* 初の制作物なので成長具合がわかるように昔のコードはあまり直さないようにしてます（制作順、暗黒竜 → 封印の剣）

---

## 使い方

1. ステータス・武器を入力
2. 支援・地形などを選択
3. 各計算結果を確認

---

## 計算仕様

* ダメージ：攻撃力 - 防御力（特効・支援・地形補正あり）
* 命中率：2RN方式を使用
* 必殺率：武器・技・支援・クラス補正を考慮
* 追撃：攻速差4以上で発生
